# DeepLearning & LLM 快速入门
个人学习路线：
- 《深度学习入门》（鱼书）约1week：快速过一遍，感兴趣的代码跟着敲一遍有个印象即可
- 《动手学深度学习》（李沐）看书+网课 约3-4weeks：推荐看书为主，重视理论推导！自己敲一遍代码！看不懂的部分去找网课对应章节听听，另外有些章节Q&A部分讲得也很好~
- 《大语言模型》(LLMBook) 人大赵鑫老师写的大模型综述，非常适合新手入门 前后断断续续看完花了约3-4weeks
